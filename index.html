Información sobre las APIs utilizadas

API de Geolocalización del Navegador

Se utiliza para obtener la ubicación del usuario (latitud y longitud) de manera nativa en el navegador.

API de Nominatim (OpenStreetMap)

Se utiliza para realizar geocodificación inversa y obtener la dirección exacta a partir de las coordenadas.

API de Clima (Weather API)

Se utiliza para obtener el clima actual y la previsión del tiempo basado en la ubicación del usuario.

WebSocket

Se utiliza para enviar y recibir mensajes en tiempo real entre los usuarios en la misma red local wifi, debe adaptarse segun la Ip del usuario que desplegara el servidor, desventaja es que no es posible desplegarla en la web de githubpages
